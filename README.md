# SQL Subqueries Employee Management

##  Project Overview
This project is a structured SQL database designed to practice and demonstrate advanced subquery concepts using a real-world Employee Management system.

The database includes multiple related tables such as Employees, Departments, Orders, Products, and Customers.

This project is ideal for SQL interview preparation and practical database learning.

---

##  Database Name
learning_subqueries

---

##  Tables Included

- Employees
- Departments
- Orders
- Products
- Customers

Each table is connected using primary and foreign key relationships to simulate real business scenarios.

---

##  Concepts Covered

- Single-row subqueries
- Multi-row subqueries
- Multi-column subqueries
- Correlated subqueries
- Aggregate functions (MAX, MIN, AVG, COUNT)
- GROUP BY and HAVING
- Foreign key constraints
- Nested queries using IN, NOT IN
- Business case SQL queries

---

##  Sample Business Queries

- Find employee with highest salary
- Find employees earning above department average
- List departments with more than 5 employees
- Find employees who have not placed any orders
- Identify products priced above average
- Find employees who placed the highest value order
- Retrieve employees with lowest salary
- Determine employees who placed the most orders
